Під час лабораторної роботи 5 було порівняно декілька регресійних та декілька класифікаційних моделей. Серед регресійних моделей найкращою виявилась лінійна регресія, а серед класифікаційних - логістична регресія без регулярізації. Хоча усі моделі дають майже однаковий результат і однозначного переможця виявити не має можливості.
